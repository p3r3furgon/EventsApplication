# EventsApplication

Этот проект представляет собой API для управления событиями, пользователями и уведомлениями, развернутый с использованием Docker.

## Запуск проекта

docker-compose up -d

## Порты сервисов

пользователи: 5001,
события: 5002,
уведомления: 5003

## Формат ввода DateTime и DateOnly:

DateTime: "2024-10-15T10:00:00Z"
DateOnly: "2024-10-15"
